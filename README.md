# Campus Hub 🚀

A peer-to-peer web platform designed to enhance collaboration among students. This platform enables students to share study materials, find peer tutors, and create study tools, addressing the gap left by official, staff-controlled college portals.

## Features

- **User Authentication:** Secure user registration and login using JWTs.
- **Course-Specific Hubs:** Organized spaces for each course to share relevant materials.
- **Material Sharing:** Students can upload and download notes, question banks, and other study resources.
- **Peer Tutoring Board:** A forum for students to ask questions and get help from their peers.
- **Student-Created Quizzes:** An interactive way for students to test their knowledge with quizzes made by other students.

## Tech Stack

**Client (Frontend):**
- React
- (Soon: Axios for API calls, React Router for navigation)

**Server (Backend):**
- Node.js
- Express.js
- MongoDB (with Mongoose)
- bcrypt.js (for password hashing)
- JSON Web Token (for authentication)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js installed on your machine
- npm (Node Package Manager)
- MongoDB Atlas account or local MongoDB instance
